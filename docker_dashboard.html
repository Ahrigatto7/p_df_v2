🐳 Docker 기반 QA 시스템 관리자

이 대시보드는 Docker 기반으로 구성된 QA 시스템을 실행하고 관리하는 도구입니다.

📦 Docker 구성 파일

  • Dockerfile - 백엔드/프론트 서버 빌드용
  • docker-compose.yml - 전체 서비스 실행 구성
  • .env - 환경 변수 설정

▶️ Docker 실행 명령

docker-compose up --build

모든 서비스(backend, streamlit, db)가 함께 실행됩니다.

🔗 링크

Streamlit 앱 열기 FastAPI API 문서 PostgreSQL DB (로컬 연결용)

📂 개발 디렉토리: backend/, frontend/, docker/, .env
